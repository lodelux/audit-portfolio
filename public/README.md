# Public Contests

Each contest gets its own folder with a short summary and links to findings.

## Suggested Structure

- `YYYY-MM-project-name/`
  - `summary.md`
  - `links.md` (optional)

## Summary Template

Use this in `summary.md` for each contest:

```
# Project Name (Platform)

- Date: YYYY-MM-DD
- Role: Solo / Team
- Result: Rank, payout, or points
- Scope: brief scope summary

## Findings

- High: X
- Medium: Y
- Low: Z
- Notes: one-line highlight about impact or unique bug

## Links

- Contest page: <url>
- My findings: <url>
- Final report: <url>
```
